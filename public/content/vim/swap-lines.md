#### Swap the current line with the previous

```
ddkP
```
  * `dd` - Cut current line
  * `k` - Move one line upwards
  * `P` - Paste before the cursor

#### Swap the current line with the next

```
ddp
```
  * `dd` - Cut current line
  * `p` - Paste
