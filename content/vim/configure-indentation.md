#### Replace tab character with spaces:

```
:set expandtab
```

#### Set the number of space characters
```
:set tabstop=2
```

#### Reindent
```
:retab
```

#### Default
```
:set expandtab
:set shiftwidth=2
:set softtabstop=2
```
